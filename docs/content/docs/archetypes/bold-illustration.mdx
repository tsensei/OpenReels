**Mood:** Energetic, clear, accessible, modern.

**Art Style:** Clean digital illustration with vibrant saturated colors. Strong outlines define shapes. Flat design elements with selective depth through layering and shadow. Modern editorial illustration style -- clean infographic aesthetics with bold, accessible visuals.

## Style Guide

**Lighting:** Flat even lighting with stylized shadows for depth. No realistic light sources -- shadows are design elements. Bold color blocks create visual hierarchy. Selective gradient fills for dimension.

**Composition Rules:** Bold centered subjects with geometric framing. Clean negative space in complementary colors. Text-free zones preserved for subtitle overlay. Strong visual hierarchy -- one dominant element per scene.

**Anti-Artifact Guidance:** Clean vector-like edges with consistent line weight. Smooth flat color fills without noise or texture artifacts. Geometric shapes with mathematically clean curves. Consistent illustration style -- every element feels drawn by the same artist.
