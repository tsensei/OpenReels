OpenReels includes 14 visual archetypes. Each archetype controls colors, caption style, motion intensity, AI image prompting, lighting, and composition rules.

## Using Archetypes

Let the AI choose the best archetype for your topic (default):

```bash
pnpm start "the fall of the Roman Empire"
```

Or specify one explicitly:

```bash
pnpm start "5 stoic lessons" --archetype warm_narrative
```

## Configuration

Each archetype is defined as a JSON file in `src/config/archetypes/`. See [Archetype Configuration](/docs/configuration/archetypes) for the full schema.
