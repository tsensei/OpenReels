**Mood:** Authoritative, fast-paced, modern, informative.

**Art Style:** Clean photorealistic renders with sharp focus and modern aesthetics. Emphasis on clarity, scale, and data-driven visual metaphors. Crisp detail, high contrast, slightly desaturated color grading with selective color pops on key subjects.

## Visual Parameters

| Parameter | Value |
|-----------|-------|
| **Background** | `#0f0f0f` |
| **Accent** | `#00d4ff` |
| **Text** | `#ffffff` |
| **Caption Style** | `box_highlight` |
| **Transition** | `slide_left` |
| **Motion Intensity** | `0.8` |
| **Font** | Space Grotesk |
| **Scene Pacing** | Fast |

**Visual Color Palette:** stark white, electric cyan, charcoal black, silver gray, deep midnight blue

## Style Guide

**Lighting:** Bright, even studio-style lighting with subtle gradient backgrounds. Clean white or cool-toned ambient light. Minimal shadows for maximum clarity. Occasional dramatic backlighting for emphasis on key subjects.

**Composition Rules:** Single-subject centered compositions with generous negative space. Clean framing, no visual clutter. Subject occupies 60-70% of frame. Strong vertical alignment for mobile viewing. Infographic-style layouts for data scenes.

**Anti-Artifact Guidance:** Sharp edges and clean lines. No blur or softness unless intentional depth-of-field. Accurate proportions on all subjects. Text-free renders. Consistent color temperature across all scenes. High-resolution detail on key subjects.

## Sample Command

```bash
pnpm start "how the internet actually works" --archetype infographic
```
